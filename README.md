This is a simple e-commerce website built using PHP, MySQL(using PDO), Bootstrap and jQuery.


**Clone the project:**

    git clone https://github.com/yons101/tp-php-ecommerce.git
    cd tp-php-ecommerce

**Configure the database:**

    config.php

**Import the script:**

    boutique.sql
   
   
**Import the data:**

    php import.php
